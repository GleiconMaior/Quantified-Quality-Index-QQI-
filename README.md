# Quantified Quality Index (QQI)

Python tools for computing Quantified Quality Index (QQI) from point clouds and estimating Rolling Resistance (RR) in mining haul roads.

## Repository structure

- src/PCA_ROUGHNESS.py – PCA-based roughness and QQI calculation  
- src/qqi_rr_gui.py – GUI for QQI → RR estimation  
- data/Calibrador do QQI_RR_GUI.xlsx – calibration dataset  

## Installation

pip install -r requirements.txt

## Usage

Run roughness analysis:

python src/PCA_ROUGHNESS.py

Run graphical interface:

python src/qqi_rr_gui.py
